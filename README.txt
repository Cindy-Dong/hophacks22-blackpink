Members: Cindy, Ashley, Linda, Belle

Descripion:
This is an user interface application for tracking work hours for PILOT leaders and beyond.
Its functions include: 
- calculate attendence percentage for a given PILOT leader on sessions;
- calculate salary for a given PILOT leader on a specific day; 
- collect work log information from users (PILOT leaders);
- calculate total salary that a PILOT leader earns;
- filter work log information for a specific leader;
- plot a leader's attendence percentage trend; 
- plot a leader's work hours trend
